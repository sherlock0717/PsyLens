# 安全与隐私（SECURITY_AND_PRIVACY）

## 凭据安全

**禁止提交**到仓库：Cookie、Token、API Key、账号密码、登录态缓存、浏览器 profile、未审查抓取缓存。

- `.gitignore` 已忽略 `.env`、cookie、缓存、`artifacts/`；
- 日志不得打印凭据；
- 真实 provider 与真实抓取默认关闭，CI 永不运行。

## 隐私边界

- 公开页面**不展示** `source_url`；公开 JSON 与 Demo 数据**不含**真实 URL（见 D-002）；
- 示例数据脱敏，无邮箱/电话/QQ/微信/账号；
- 完整数据下载默认**不在页面提供**（见 D-001）；
- 不做大规模再分发（平台条款风险，见延期决策）。

## 平台条款

抓取需遵守目标平台使用条款、robots 规则与频率限制。相关风险已登记为 `requires_decision`（见隐私审计与决策包）。

## 数据主体

样本为公开社区反馈节选。任何潜在可识别信息在公开层移除或脱敏；底层历史数据不删除但不公开下载入口。
