# Prompt 模板：证据抽取（evidence_extraction）

> 状态：`reconstructed_template`。公开模板，不保证与历史运行逐字一致。

## 目标

把一条反馈原文切成若干**可单独判断的证据单元**，每个单元是原文的连续片段。

## 指令（模板）

```
把下面的反馈切成若干证据单元。要求：
1. 每个单元是原文的连续子串，不要改写；
2. 一个单元只表达一个相对完整的抱怨/观点；
3. 纯语气词、上下文依赖的中间句可不切出。
输出 JSON 数组：[{ "unit_index": 1, "unit_text": "原文子串" }, ...]
反馈：{{cleaned_text}}
```

## 校验

`unit_text` 必须能在原文中定位（子串匹配）；否则丢弃该单元。
