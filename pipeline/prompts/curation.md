# Prompt 模板：反馈精修（curation）

> 状态：`reconstructed_template`。这是根据现有脚本调用结构重建的**公开模板**，
> 不保证与最初历史运行逐字一致。真实调用需自行提供模型与 API Key（不入库）。

## 目标

对合并后的原始反馈做清洗与精修：去除噪声、保留可分析的完整语义单元，不改变原意。

## 输入

单条反馈原文（`raw_text`）与其平台、时间等元数据。

## 指令（模板）

```
你是社区反馈清洗助手。请对下面这条反馈：
1. 去除与游戏/产品体验无关的纯灌水、表情、广告；
2. 保留完整语义，不要改写观点、不要补充信息；
3. 如果整条无可分析内容，标记 keep=false。
输出 JSON：{ "keep": true|false, "cleaned_text": "...", "reason": "..." }
反馈：{{raw_text}}
```

## 输出

`{ keep, cleaned_text, reason }`。`keep=false` 的条目不进入后续采样。
